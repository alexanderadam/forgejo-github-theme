README.md

hello-world

A tiny greeting-server used as the working example in the onboarding docs. Accepts a name via a query parameter and returns a polite wave in whatever language the client's Accept-Language header asks for — or English if nobody asked.

Quick start

pip install -e .
python -m hello_world --port 8080

Then curl it:

curl 'http://localhost:8080/?name=Alex'
# => Hi Alex, waves politely.

Why does this exist?

Mostly so the onboarding docs have a repo they can link to that's small enough to read in full, but exercises routing, templating, i18n, and CI. If you're looking for a real HTTP framework, start somewhere else.

Contributing

PRs welcome. Keep the diff small and the commit messages honest. Tests in tests/, pytest runs them.
