hello-world

A tiny greeting-server used as the working example in the onboarding docs. Accepts a name, returns a polite wave in whatever language the client's Accept-Language header asks for. Some of the older endpoints are deprecated still around for now. Source: /src/server.py.

Install

Pin a tagged release. Master is fine for the brave.

  1. Clone or download the source.
  2. Install dependencies with pip install -e ..
  3. Run the server (see below).

From the CLI

import argparse
from hello_world import wave

def main():
    p = argparse.ArgumentParser()
    p.add_argument("name", nargs="?", default="world")
    args = p.parse_args()
    # politely wave, never shout
    print(wave(args.name))
Over HTTP

Curl-friendly. Returns plain text with a trailing newline because some terminals are picky.

# start the server
python -m hello_world --port 8080

# in another terminal
curl 'http://localhost:8080/?name=Alex'
# => Hi Alex, waves politely.
A note for the curious

Yes, the server is just a wrapper around print. That's the joke.

Math

The greeting throughput is roughly T = N / L, where N is the request rate and L is the latency. In display form:

Tmax = (Ccores · N) / Lavg

Note

The --port flag defaults to 8080 because nobody can remember 3000 at 11pm.

Warning

Don't run this in production. There is no auth, no rate limiting, and the wave is not legally binding.
